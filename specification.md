# 🐾 Rescue Cat Life（保護猫ライフ）仕様書 v1

## 1. 概要

「Rescue Cat
Life」は、ユーザーが保護猫を引き取り、日々の「お世話」を通じて徐々に心を開いていく過程を楽しむWebベースの育成シミュレーションゲーム（MVP）です。\
最終的に信頼関係を築き上げ、新しい家族（ずっとの家族）として迎え入れる「譲渡エンディング」を目指します。

------------------------------------------------------------------------

# 2. コア体験（目的）

-   保護されたばかりの警戒心の強い猫が、お世話を通じて少しずつ信頼を寄せてくる過程を体験する
-   最終目標（エンディング）：Trust（信頼度）を100に到達させる

------------------------------------------------------------------------

# 3. ステータス（主要パラメータ）

猫の状態は以下の4つで管理する。

  ステータス           範囲     説明
  -------------------- -------- --------------------------
  Hunger（空腹）       0〜100   高いほどお腹が空いている
  Stress（ストレス）   0〜100   高いほど警戒している
  Dirty（汚れ）        0〜100   環境やトイレの汚れ
  Trust（信頼）        0〜100   猫の懐き度

初期値\
- Trust = 10

エンディング条件\
- Trust = 100

------------------------------------------------------------------------

# 3.1 Tick処理（時間経過システム）

現実の **15分を1Tick** とする。

放置するとステータスが悪化する。

### 通常時（Sleep Window外）

-   Hunger +4 / Tick
-   Dirty +3 / Tick
-   Stress +2 / Tick
-   Dirtyが60以上のとき Stress追加悪化

※数値は旧仕様のままとする

------------------------------------------------------------------------

# 3.2 クライシス状態

以下のどれかが80以上

-   Hunger
-   Stress
-   Dirty

この状態を **クライシス（Crisis）** と呼ぶ。

### クライシス状態でTickが発生すると

Trust -1

ただし

Sleep Window中はTrustは減少しない

------------------------------------------------------------------------

# 3.3 睡眠システム（Sleep Window）

ユーザーが設定する睡眠時間。

ユーザーは

-   就寝時間
-   起床時間

を設定できる。

例

23:00 〜 07:00

この時間を **Sleep Window** と呼ぶ。

------------------------------------------------------------------------

### Sleep Window中のTick

-   Hunger +0
-   Dirty +0
-   Stress -1
-   Trust減少なし

この時間はユーザーがプレイできない時間として扱う。

------------------------------------------------------------------------

# 4. プレイヤーアクション（お世話）

  アクション   Hunger   Stress   Dirty   Trust
  ------------ -------- -------- ------- -------
  ごはん       -20      -3       0       +1
  遊ぶ         +6       -18      0       +2
  掃除         0        -5       -25     0
  休む         +4       -12      0       +1

※Trust15未満の時 遊ぶ → Trust +1

# 5. UI構成

## CARE

## LOG

-   信頼イベント

------------------------------------------------------------------------

# 6. 信頼イベント

Trustが一定値に到達するとイベント発生

例

-   5 : 少し近づいてきた
-   10 : 匂いを嗅いできた
-   20 : 足元に来た
-   50 : 膝に乗る

ログに保存される

------------------------------------------------------------------------

# 7. エンディング

Trustが100になると

「ずっとの家族」エンディング

猫は新しい家族へ譲渡される。

------------------------------------------------------------------------

# 8. 卒業アルバム

エンディング後

新しい猫を迎えることができる。

過去の猫は履歴に保存される。

保存内容

-   名前
-   毛色
-   卒業日
